Objectives
Our department SharePoint site is really coming along! The team has asked you to improve the menu system to make content easier to find. The solution: mega menu!

In this exercise, you will

  • Configure navigation settings
  • Design the mega menu appearance
  • Create CSS attributes
  • Build the mega menu
Configuring navigation settings
1.
If not already open, open Internet Explorer and go to your department's site.
http://[island].msc.com/sites/[department]
2.
Click Settings | Site Settings.
3.
Under Site Actions, click Manage site features.
4.
Scroll down the page to SharePoint Server Publishing. Click Activate.
5.
Go back to the Site Settings page.
6.
Under "Look and Feel," select Navigation.
7.
On the Navigation Settings page, change the Global Navigation to Structural Navigation.
8.
Select the box next to Show subsites and deselect Show pages.
9.
In the Current Navigation area, select Structural Navigation: Display only the navigation items below the current site.
10.
Scroll down the page to the Structural Navigation: Editing and Sorting section.
11.
Select Global Navigation, then click Add Heading.

24.
One of the simplest methods for adding script to a SharePoint page is to use the Content Editor Web Part. Go back to your site's home page and click EDIT in the upper right corner.
25.
Click above the Local Weather app, then select the INSERT tab from the ribbon.
26.
Select Web Part from the ribbon.
27.
From the Categories menu, select Media and Content.
28.
Click the Content Editor in the Parts menu, then click Add.
29.
Open the Edit Web Part properties pane for the Content Editor Web Part.
30.

Edit the Content link by clicking the ellipsis and entering:

 

http://[island].msc.com/sites/[department]/SiteAssets/Menu.txt

31.
Expand the Appearance section of the Content Editor properties and set the title to Mega Menu
32.
Change the Chrome Type to None and click OK at the bottom of the Properties pane.
33.
Save your changes by clicking Save in the ribbon.
34.
Refresh the page and hover over the Regional Departments and Useful Links items in the Global Navigation.
Cool!
This method works well at an individual site level, but it is not recommended for multiple sites or public-facing SharePoint implementations.
35.
This is nice. Let's make it better! Open your site in Designer if it is not already open.
If Designer was already open, click the Refresh button in the Quick Access Toolbar to refresh the site contents.
36.
Select All Files from the Site Objects pane.
37.
Click Site Assets.
38.
Open Menu.txt.
39.
On the Menu.txt Designer page, click Edit File in the Customization section.
40.
Scroll down to the bottom of the file to the CSS tag called UL.featuredNavigation on line 86.
41.
Place your cursor at the end of the width attribute, line 88, and press <Enter> to make a new line.
42.

To smooth out the corners, verify the following CSS3 attribute:

 

border-radius:10px;

43.
Save the script.
Your changes are saved and immediately visible in SharePoint.
44.
Leaving Designer open, switch back to your home page in Internet Explorer and refresh the page.

Hover over the Regional Departments. The mega menu should now look like this:
46.
Almost there. We can make one more CSS change to really make our menu smooth. Go back to Designer.
47.

Add a new line after the border-radius attribute and type the following:

 

opacity:0.95;

Feel free to adjust the opacity. 95 percent is a nice balance.
48.
Again, save your work. Switch back to your home page in Internet Explorer and refresh.
