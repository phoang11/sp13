Working With Site Columns

Objective
Before MSC began to expand, it was simple to keep track of who was on vacation or just taking a much-needed day off. Now that we have grown, we need a better way to keep track of who’s in and who’s out.

In this exercise, you will

  • Create site columns for the Time Off content type
Creating site columns for the Time Off content type
1.
Open your department site in SharePoint Designer 2013.
You can open your site by double-clicking it from the top of the Recent Sites list in Designer.
2.
Click Site Columns in the Site Objects pane.
3.
Create the first site column by clicking New Column in the New section of the ribbon and selecting Choice.
4.

Enter the following in the "Create a Site Column" dialog box and click OK:

 

Name: Time Off Type

Description: Type of time off request

New Group: _[Department]

An underscore prefix was added to the group name so that your group will appear at the top of the list of site columns.
5.

Enter the following in the Column Editor dialog:

 

Choices:Vacation
 Personal
 Sick
Default value:Vacation
Other:Clear Allow blank values?
6.
Click OK.
7.
Save your work by clicking the Save button or pressing <Ctrl><S>.
You should now have one site column listed under your _[Department] group.
8.
Add another column by clicking New Column in the ribbon, and select Number from the New Column pull-down menu.
9.

Enter the following in the "Create a Site Column" dialog box:

 

Name: Time Off Hours

Existing group: _[Department]

10.
Click OK.
11.

Using what you have just learned, create the following site columns and add them to the existing site column group _[Department]:

 

NameType
Time Off CommentMulti-Lines of Text
Time Off Start DateDate & Time
Time Off Return DateDate & Time
We are prefixing each site column name with the words "Time Off" so that they are easy to locate when we use them in content types.
12.
Click Save to save your work. Your Site Columns tab should look like this:

We need to change some settings for the site columns we just created, by selecting the column and clicking Column Settings in the Edit section of the ribbon.

 

You can also edit column settings by double-clicking the column name.

13.
Configure the Time Off Comment site column's settings by double-clicking Time Off Comment in the Site Columns list.
14.
In the Column Editor dialog, make sure the checkbox for "Rich text" is cleared.
15.
Click OK.
16.
Double-click Time Off Hours in the Site Columns list.
17.
Clear the "Allow blank values" checkbox.
18.
Set the "Number of decimal places" to None (whole numbers).
19.
Click OK.
20.
Open the Time Off Start Date site column for editing.
21.
In the Column Editor, clear "Allow blank values."
22.
Click OK.
23.
Open the Time Off Return Date site column for editing.
24.
In the Column Editor, clear "Allow blank values."
25.
Click OK.
26.
Click Save to save your work.
27.
Close the Site Columns tab in Designer.
Congratulations! You have created new site columns for the Time Off Request content type to be developed in the next exercise.

This is the end of the exercise.
