Objectives

 

Project managers have asked for a way to recall a project budget submission in case they need to make a change before or during the Manager Approval Cycle.

 

In this exercise, you will

  • Create the project manager and manager approval process
  • Improve the review cycle between the project manager and the manager using a workflow loop
