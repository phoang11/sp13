Objective
The salespeople from each department want a quick visual indicator to let them know which customers are spending less this quarter than they did last quarter.

In this exercise, you will

  • Disable Minimal Download Strategy (MDS)
  • Reformat the currency columns
  • Add up and down arrows based on this quarter's performance

The overriding functions are fairly simple. The formula is:

  1. Declare a local variable and set it equal to the current field value.
  2. Return a reformatted/enhanced value in its place.

       Number(ctx.CurrentItem.CurrentQtrRevenue);

This function will place a dollar sign in front of the Current Quarter Revenue and reformat the number so that only two decimal places appear.
