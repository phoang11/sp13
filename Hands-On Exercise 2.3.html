Hands-On Exercise 2.3
Creating and Using Content Types
Exercise list …








Objectives

In this exercise, you will

  • Build the Time Off content type
  • Set up a list with the new content type
Building the Time Off content type
1.
Open your SharePoint site if Designer is not already open.
2.
Select Content Types from the Site Objects pane.
3.
Click Content Type in the New section of the ribbon to create a new content type.
4.

In the "Create a Content Type" dialog box, enter the following:

 

Name: Time Off Request
Description: Create a time off request
Select parent content type from: List Content Types

Select parent content type: Item

New group: _[Department]

5.
Click OK.
You do not need to save your work when creating a new content type. SharePoint Designer saves the content type when you click OK in the Create dialog box.
6.
Click the Time Off Request content type you just created in the Content Types list.
7.
On the Time Off Request content type tab in Designer, click Edit content type columns in the Customization section.
8.
Click Add Existing Site Column from the New section of the ribbon.
9.
From the Site Column Picker dialog, double-click the Time Off Type site column.
You can easily find your site columns by typing Time in the search field.
10.

Add the remaining Time Off columns by repeating the above steps in the following order:

 

Time Off Hours

Time Off Comment

Time Off Start Date

Time Off Return Date

If your Add Existing Site Column button is disabled in the ribbon, click anywhere in the Columns editor.
11.
Click Save to save your work.
Every list and library in SharePoint has a Title column. However, we don't need it for our Time Off Request list. We cannot delete it, but we can hide it.
12.
Click the Property field of the Title column. After a moment, a pull-down menu will appear.
13.
Click the down arrow to open the pull-down menu.
14.
Select Hidden from the list. Your site columns list should look like this:
15.
Click Save to save your work.
16.
Still in SharePoint Designer, click Lists and Libraries in the Site Objects pane.
17.
From the New section of the ribbon, click Custom List.
18.

In the "Create list or document library" dialog, enter the following:

 

Name: Time Off Requests

Description: Employee time off requests

19.
Click OK.
20.
Open the Time Off Requests list by clicking the list name in the "Lists and Libraries" tab.
21.
In the Settings section, select Allow management of content types and Display this list on the Quick Launch. Clear "Allow attachments."
Click OK to accept the Designer warning when clearing the "Allow attachments" checkbox.
22.
Add the Time Off Request content type by clicking the Add button in the Content Types section.
23.
In the Content Types Picker dialog, select the Time Off Request content type.
24.
Click OK.
25.
Select the new content type by clicking to the right of the content type title.
26.
Click Set as Default from the ribbon.
27.
Select the Item content type using the same technique for selecting the Time Off Request in a previous step.
28.
Click Delete in the ribbon.
29.
A Confirm Delete dialog will appear. Click Yes.
30.
Add a new default view for the Time Off Request list by clicking New in the Views section.
31.
Name the new list view All Requests
32.
Set this new view as the default view by selecting the checkbox Make this the default view.
33.
Click OK.
34.
Click Preview in Browser from the ribbon.
35.
In Internet Explorer, select new item to add a new request.
36.

Enter the following test values for the new time-off request:

 

Time Off Type: Vacation
Time Off Hours: 16

Time Off Comment: Long weekend
Time Off Start Date: [This Friday]
Time Off Return Date [The following Tuesday]

37.
Click Save.
38.
Customize the All Requests view by clicking All Requests, then clicking the ellipsis (. . .) to the right of All Items.
39.
Click Modify this View.
40.
On the Edit View page in the Columns section, clear the Title column.
41.

Still in the Columns section, select the following columns and set the order:

 

Column NamePosition From Left
Created By1
Time Off Hours2
Time Off Comment3
Time Off Start Date4
Time Off Return Date5
42.
In the Sort section of the Edit View page, change the "First sort by the column" pull-down to ID.
43.
Change the sort order by selecting the Show items in descending order radio button.
44.
Scroll down the page to the Group By section.
45.
Expand the Group By section by clicking the small plus sign.
46.
Change the "First group by the column" field to Time Off Type.
47.
Still in the Group By section, change the default groupings to Expanded.
48.
Scroll to the bottom of the page and click OK.
The All Requests view should look something like this:
Congratulations! You have created and tested a content type for time-off requests.

This is the end of the exercise.
