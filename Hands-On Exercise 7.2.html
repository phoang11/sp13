Link External Data With Workflow Logic

Objective
For improved efficiencies and some paperwork reduction, the HR manager has requested that the Time Off workflow interact with the HR data to check the number of available hours per employee.

In this exercise, you will

  • Integrate the Time Off Request Process with the HR database to check availability
Integrating the Time Off Request Process with the HR database to check availability
Unlike SharePoint Designer 2010, SharePoint Designer 2013 does not support access to an external list from workflows. We are going to work around this limitation by copying the external list data to a site list.
1.
Select Lists and Libraries from the Site Objects pane.
2.
Click Custom List in the ribbon.
3.

Set the following values in the "Create list" or "document library" dialog:

 

Name: Available Hours Copy

Description: List for linking external HR data to Time Off Request process

4.
Click OK.
5.
Click Available Hours Copy in the Lists list.
6.
Click Edit list columns in the Customization section.
7.
Rename the Title field to AccountName
8.
Click Add New Column in the ribbon.
9.
Select Number from the menu.
10.
Change the column name to VacationHours
11.
Add the following columns by repeating the two previous steps:
NameType
SickHoursNumber
PersonalHoursNumber
12.
Click Save to save your work.
13.
Select Workflows from the Site Objects pane.
14.
Click Site Workflow in the ribbon.
15.

Set the following values in the Create Site Workflow dialog:

 

Name: Copy External List Available Hours

Platform Type: SharePoint 2010 Workflow

16.
Click OK.
17.
Rename Step 1 Initialize Variables. Place the orange cursor in the Initialize Variables step.
18.
Add a Set Workflow Variable action.
19.
Click the workflow variable link.
20.
Select Create new variable.
21.

Set the following values in the Edit Variable dialog:

 

Name: Department

Type: String

22.
Click OK.
23.
Click the value link.
24.
Type [Island].[Department] (this is case sensitive!).
25.
Press <Enter>.
26.
Place the orange cursor below the Initialize Variables step.
27.
Create a new step.
28.
Rename the new step Copy External List
29.
Place the orange cursor within the Copy External List step.
30.
Add a Create List Item action.
31.
Click Save to save your work.
32.
Click the this list link.
33.
Set the List to Available Hours Copy in the Create New List Item dialog.
34.

35.
Click the Modify button.
36.
Click the fx button in the Value Assignment dialog.
37.
In the "Lookup for Single line of text" dialog, change the "Data source" value to Available Hours.
38.
Set the "Field from source" to Account Name.
39.
Change the Field value in the "Find a List Item" section to AccountName.
40.
Click the fx button next to the Value field.
41.

Set the following values in the "Lookup for Single line of text" dialog:

 

Data Source: Workflow Variables and Parameters

Field from source: Variable: Department

Starting the Available Hours Copy workflow
90.
Open your site in Internet Explorer.
91.
Click Site Contents in the Quick Launch.
92.
Select SITE WORKFLOWS in the upper right corner of the page.
93.
Click Copy External List Available Hours to start the site workflow.
94.
Click Start.
The site workflow will start.
95.
Confirm that the site workflow has completed successfully. The Internal Status will read "Completed."
96.
Click Available Hours Copy from the Quick Launch.
Your Available Hours Copy should now have the same item as the Available Hours external list.
97.
Close Internet Explorer.
Congratulations! You have transferred external list data to a SharePoint custom list. This will enable our Time Off Request workflow to use HR data!
