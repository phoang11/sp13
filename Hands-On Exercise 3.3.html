Check your inbox in Outlook. You will receive an e-mail from SharePoint assigning you a new task to review the time-off request you just added.
75.
In the e-mail, click the hyperlink Time Off Request: Please Review to go to the assigned task.
Internet Explorer will take you to the Time Off Request assignment. Assignments are tasks stored in the Time Off Request Process Tasks list.
76.

77.

78.

79.
On the Time Off Request edit page, click the Approved button to approve the request.
The Time Off Request task will show as complete in the task list.
80.
