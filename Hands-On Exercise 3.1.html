Hands-On Exercise 3.1
Introduction to Workflows in SharePoint Designer
Exercise list …








Objective
Your department at Maui Shores Consulting (MSC) needs to move all confidential documents to a special document library. You will create a quick workflow to automate this requirement.

In this exercise, you will

  • Build a workflow for a document library to move documents with the word Confidential in the title
Create the Confidential document library
1.
If not already open, start SharePoint Designer and open your department site.
2.
Select Lists and Libraries from the Site Objects pane.
3.
Click the Document Library button in the ribbon.
4.
Select Document Library from the menu.
5.

Fill in the following information in the Create list or document library dialog box:

 

Name: Confidential Documents

Description: Library to store confidential documents

6.
Click OK.
Create the workflow to move confidential documents
7.
Select Workflows in the Site Objects pane.
8.
Click the List Workflow button in the ribbon.
9.
Select Documents from the List Workflow menu.
10.

In the Create List Workflow | Documents dialog, enter the following:

 

Name:Move Confidential Documents
Description:Workflow to move documents with the word Confidential in the title to a separate document library.
Platform Type:SharePoint 2013 Workflow
11.
Click OK.
12.
Click Workflow Settings in the Manage section of the ribbon.
13.
In the Start Options section, select the checkbox next to Start workflow automatically when an item is created.
14.
Click Save in the ribbon to save the workflow.
15.
Select the Edit Workflow button in the Edit group of the ribbon.
16.
Click the word Stage 1 in the first stage of the workflow.
17.
Rename the stage to Move Confidential Document
18.
Place the orange cursor in the stage by clicking just below the phrase "(Start typing or use. . ."
19.
Click Action in the Insert section of the ribbon.
20.
Select Find Substring in String from the menu.
21.
Click the substring link.
22.
Type the word Confidential
23.
Press the <Enter> key.
24.
Click the string link.
25.
Click the fx button.
26.
In the Lookup for String dialog, change the "Field from source" value to Name.
27.
Click OK.
28.
Place the orange cursor below the Find action from the previous steps.
29.
Click the Condition button in the ribbon.
30.
Select If any value equals value from the menu.
31.
Click the first value link in the condition you just added.
32.
Select the fx button.
33.

In the Define Workflow Lookup dialog, set the following values:

 

Data source: Workflow Variables and Parameters

Field from source: Variable: index

34.
Click OK.
35.
Change the equals value to is greater than.
36.
Click the last value link.
37.
Type 0 (the number zero).
38.
Press the <Enter> key.
39.
Place the orange cursor below the word Start within the condition added in the previous steps.
40.
Type the word copy and press <Enter>.
Typing the first word in an Action or Condition is a quick way to build your workflow.
41.
Click the first this library link.
42.
In the Choose List Item dialog, confirm the List value is set to Current Item.
43.
Click OK.
44.
Click the second this library link.
45.
Set the value to Confidential Documents.
46.
Place the orange cursor beneath the Copy action added in previous steps.
47.
Type the word Delete and press <Enter>.
48.
Click the this list link.
49.
Confirm that the List value is set to Current Item.
50.
Click OK.
51.
Place the orange cursor in the Transition to stage section.
52.
Type Go and press <Enter>.
53.
Click a stage in the "Go to a stage" action.
54.
Select End of Workflow from the menu.
Your workflow should now look like this:
55.
Click Publish in the Save section of the ribbon to publish the workflow.
Test the workflow
56.
Select Lists and Libraries in the Site Objects pane.
57.
Click Documents in the Document Libraries group.
58.
Click the Preview in Browser button in the ribbon.
59.
Click the new document link in Internet Explorer.
60.
In the "Add a document" dialog, click the Browse button.
61.
Go to C:\Course Files and select the Privacy Policy – Confidential Word document.
62.
Click Open in the Choose File to Upload dialog.
63.
Click OK in the "Add a document" dialog.
64.
Click Confidential Documents in the Recent group of the Quick Launch.
You should now see the HIPAA document in the Confidential Documents library. It may take a up to minute for the workflow to move the document. If you do not see the document, refresh the page.
65.
Click the Documents library in the Quick Launch.
The Documents library should now be empty.
Congratulations! You have created your first workflow in SharePoint 2013.

This is the end of the exercise.
